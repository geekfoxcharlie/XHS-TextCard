# XHS-TextCard | 小红书智能文字卡片工具

**XHS-TextCard** 是一款专为小红书创作者打造的智能卡片生成工具。它不仅能将您的**深度长文**一键转化为排版精美的系列图片，更致力于通过**全方位的视觉定制**（专属签名、防盗水印、自定义模版），助你打造**独一无二的个人 IP 视觉风格**。

## 🎯 解决的核心痛点

小红书创作者常遇到的痛点：
- **手动分页效率低下**：长文发布需要手动分割成多张图片，耗时耗力
- **排版风格不统一**：每张图片的视觉效果参差不齐，影响专业形象
- **缺乏个人品牌标识**：内容容易被抄袭，难以形成独特辨识度
- **参数调节繁琐**：频繁调整字体、颜色、间距，创作流程不够流畅
- **在线工具使用壁垒**：多数工具需要注册登录，受网络影响创作不自由
- **精细定制收费高**：好的模板和高级功能往往需要付费，增加创作成本
- **隐私数据安全隐患**：在线工具可能泄露创作内容，存在版权风险

## 🌟 核心功能：智能文字转卡片

这是本项目的灵魂引擎，集智能排版与视觉定制于一体：

- **智能分页引擎**：内置 TextSplitter 算法，精准计算文字高度与段落结构，自动将长文拆分为多张 3:4 黄金比例图片，彻底告别手动分页的烦恼。
- **实时预览同步**：左侧输入文字，中间实时预览，右侧即时调参。300ms 防抖优化，确保流畅的编辑体验。
- **智能 Markdown 解析**：
    - **多级标题支持**：自动识别 #、##、### 语法并美化显示
    - **文本格式化**：支持 **加粗**、==高亮== 等基础语法，让文字更有层次感
- **大师级预制模板**：内置 6 款深度适配小红书审美的模板（极简杂志、苹果备忘录、奶油拿铁、弥散极光、暗夜深思、大厂文档），一键切换，秒变设计大师。
- **深度视觉定制**：
    - **排版微调**：字号、行高、字间距、内边距均可像素级调节
    - **字体选择**：内置苹方、微软雅黑、宋体、楷体、黑体等多种字体
    - **色彩管理**：内置莫兰迪色系等多款配色预设，支持 HEX/RGBA 及渐变色背景自定义
    - **背景模式**：支持纯色背景与线性渐变背景的自由切换，可调节渐变角度（0-360°）
- **一键批量导出**：自动生成高清图片（3:4 黄金比例，适配小红书最佳显示效果），支持单张下载或一键打包为 ZIP 文件，出图效率提升 10 倍。

## 🚀 打造专属 IP 视觉

我们深知“辨识度”对于创作者的重要性，因此提供了强大的专属性配置：

### 1. 品牌签名栏 (Signature System)
- **多风格选择**：内置 Terminal（极客风）、现代胶囊、优雅衬线、极简毛玻璃等多种设计风格。
- **内容自定义**：支持自定义签名文案（如 @你的账号）及专属品牌色。

### 2. 版权防盗水印 (Watermark)
- **背景保护**：支持开启背景水印，有效标识内容原创性。
- **隐形嵌入**：可精细调节水印颜色的透明度，在宣誓主权的同时，确保不影响读者的阅读体验。

### 3. 个人模板库 (Template Library)
- **自定义保存**：调配出满意的参数组合后，可将其**保存为自定义模板**（导出为配置及样式文件）。
- **方便复用**：将生成的模板文件放入项目目录，刷新后即可在模板列表中随时调用，确保你发出的每一篇笔记都拥有统一的视觉语言。

### 🛠️ 模板扩展指南（开发者）

项目采用**分层模板架构**，开发者可轻松添加新模板：

```
templates/
├── index.json              # 模板索引
├── {template-id}.json      # 模板配置（名称、配色、默认参数）
└── {template-id}.css       # 模板样式（签名栏、水印等视觉效果）
```

**添加新模板步骤：**
1. 在 `templates/` 目录创建 `{id}.json` 和 `{id}.css`
2. 在 `index.json` 中注册新模板
3. 刷新页面即可在模板列表中看到新模板

模板系统支持丰富的视觉定制，包括背景样式、签名栏风格、水印效果等。

## 🛠️ 技术特性

- **纯本地运行**：项目基于现代前端技术构建，**0 服务器交互**，依赖少量外部库（dom-to-image-more、jszip、marked 等）提供增强功能。
- **隐私绝对安全**：所有的文字草稿、图片素材、模板数据均只存在于你的浏览器本地，绝不会上传至任何云端，放心创作。
- **离线可用**：下载项目包后，即使在无网络环境下（如飞机、高铁上）也能流畅使用（首次使用需联网加载外部库）。
- **极简轻量**：无复杂的构建流程，解压即用，占用极少系统资源。
- **实时预览优化**：采用防抖算法和高效的DOM渲染，确保大文本也能流畅预览。
- **智能分割算法**：基于字体度量和容器尺寸的精确计算，确保分页准确性和美观性。

## 💡 使用技巧

- **智能分段**：用两个回车 `↵↵` 分隔段落（推荐），系统会优先在段落间分页，避免页面末尾出现孤立的单行文字
- **Markdown 语法**：善用 `# 标题`、`## 二级标题`、`**加粗文字**`、`==高亮文字==` 来丰富内容层次
- **品牌一致性**：通过自定义模板保存你的专属参数组合，确保系列内容视觉风格统一
- **水印调节**：透明度设置为 0.1-0.2，既能保护版权又不影响阅读体验
- **渐变角度**：135° 通常是最舒适的阅读角度，可根据内容主题适当调整

## 🚀 快速开始

1. **下载项目**：获取项目源码包并解压到任意目录。

2. **启动本地服务器**（推荐方式）：
   ```bash
   # Python 3.x
   python -m http.server 8000

   # Python 2.x
   python -m SimpleHTTPServer 8000

   # Node.js (如果安装了)
   npx http-server -p 8000

   # PHP (如果安装了)
   php -S localhost:8000
   ```

 3. **打开浏览器访问**：
    - 文字转卡片：http://localhost:8000/

> **注意**：不推荐直接双击HTML文件打开，因为浏览器的安全策略可能导致部分功能无法正常工作。
