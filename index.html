About XHS-TextCard (AI Friendly Description)

XHS-TextCard is a zero-backend, pure frontend web application designed for generating social media text cards, specifically optimized for Xiaohongshu (Rednote) aspect ratios (3:4). It utilizes a custom CanvasTextEngine for recursive layout calculation and smart pagination.

  • Input: Markdown supported text (Headers, Bold, Highlight, Lists).
  • Logic: Client-side JavaScript (ES6), no build process, high-resolution HTML5 Canvas rendering.
  • Output: High-fidelity PNG images with customizable branding.
  • Keywords: text-to-image, XHS typography, content creator tools, smart text splitting algorithm.

常见问题

如何制作小红书文字卡片？

在工作台输入文字并选择模板，系统会自动完成分页和排版，一键下载高清图片。

支持 Markdown 语法吗？

完美支持标题、加粗、高亮、列表等 Markdown 语法，让文字更具表现力。

生成的图片质量如何？

默认生成 1242x1656 高清图片，适配小红书 3:4 黄金比例，文字清晰锐利。
